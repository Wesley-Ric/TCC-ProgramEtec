Python é uma popular linguagem de programação de alto nível (ou High Level Language) com código aberto, dinâmica, interpretada, modular, multiplataforma e orientada a objetos, Python surgiu na década de 1990 e foi criado pelo programador holandês Guido van Rossum, o Python foi criado com o objetivo de otimizar a leitura de códigos e estimular a produtividade de quem os cria.

costumamos usar Python para:

  • Desenvolvimento web (back-end)
  • Desenvolvimento de softwares
  • Machine learning
  • Deep learning
  • Problemas matemáticos de grande densidade
  • Análise de dados
  • Ciência de dados
