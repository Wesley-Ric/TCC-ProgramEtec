Por que HTML?

HTML foi criado na decada de 90 pelo cientista Tim Berners-Lee, na Suíça com a finalidade de tornar possível o acesso e a troca de informações de pesquisas, entre cientistas de diferentes universidades. No ano de 1992 foi liberada a biblioteca de desenvolvimento WWW(World Wide Web), rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB. HTML(Linguagem de Marcação de Hipertexto) é uma linguagem utilizada para construção de páginas

O HTML tem como função criar o corpo da sua Página, é a linguagem principal da web, ela permite que desenvolvedores desenhem como serão exibidos pelo navegador os elementos da página, como arquivos multimídia, texto e hiperlinks.
