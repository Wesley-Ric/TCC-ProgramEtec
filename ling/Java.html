O que é Java?
Utilidades
Vantagens de se aprender
O Java foi criada e comercializada pela Sun Microsystems em 1995 e nos dias de hoje se encontra mantido pela empresa Oracle. O Java é conhecido como uma linguagem de programação orientada a objetos que é muito usado para o desenvolvimento de aplicativos e softwares.

Ao contrário do que muitos pensam, Java não tem nenhuma ligação com o JavaScript, o JavaScript possui esse nome devido apenas a popularidade do Java, que na época, era a linguagem mais utilizada no mundo todo.
