Variáveis
O que são variáveis?
Declaração
Constantes

Agora que já sabemos o que é uma varíavel, vamos aprender a como cria-las, mas antes disso, precisamos aprender quais são os tipos e para que serve cada um deles.

Tipo de variável Função
int Números inteiros
Double ou Float Números decimais
String Caracteres não calculáveis

Nessa tabela deixei apenas os mais usados, mas existem muitos outros tipos de variáveis de diferentes tamanhos e fins.

Nesse momento, nós focaremos em ver os tipos de variável int e double, sendo o int, os números sem vírgula (inteiros), e double com números com vírgula(decimais).

Segue um exemplo:

public class PrimeiraClasse{

public static void main(Strings[] args) {
int a; //Número Inteiro
double b; //Número decimal
String c; //Caractere
}

}

Na programação acima, declaramos 3 varíaveis de 3 diferentes tipos, int, double e string, respectivamente.

Vale dizer que também existe uma regra para a nomeação de variáveis, são basicamente as mesmas das classes, mas agora não é necessário ter a primeira letra em maiúsculo, além disso, também é possível separar 2 palavras usando hífen.

EX: int placa-carros;
