if e else

Nas linguagens de programação, também existem o que chamados de estruturas de decisão, os famosos if e else, elas servem para tomar decisões dentro da nossa programação tendo como base um parametro imposto por nós.

  • if - se

  • else - se não

Esse tipo de estruta é bem simples de se usar, funciona como se fosse um "questionamento"

Se (tal coisa for verdade) {
faça tal coisa;
}


Senão {
faça outra coisa;
}

As estruturas de decisão funcionam dessa forma, agora basta apenas vermos a sintaxe.

if (idade>=18){
System.out.println(" Maior de idade ");
}

else{
System.out.println(" Menor de idade ");
}

Na estrutura de decisão acima, estamos dizendo que, IF(se) a variávelidade for MAIOR OU IGUAL a 18, aparecerá no prompt uma mensagem dizendo que o usuário é maior de idade, else (caso contrário), aparecerá que o usuário é de menor.

Além de IF's e ELSE's, nós também temos uma terceira condicional, o else if, que funciona como um "também se", ou seja, é mais uma condição if dentro da estrutura.

Segue um exemplo de utilização do else if

if (nota>7){
System.out.println("Sua nota está acima da média");
}

else if (nota=7){
System.out.println("Sua nota está na média");
}

else{
System.out.println("Sua nota está abaixo da média");
}
