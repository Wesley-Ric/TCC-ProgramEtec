if e else

Na nossa primeira operação matemática, somaremos 2 constantes e mostraremos o resultado no prompt de comando, segue a programação:

int valor1 = 8; // declaração da primeira constante

double valor2 = 7.5; // declaração da segunda constante

double resultado = valor1+valor2; // criação da variável de soma das duas constantes

System.out.println("O resultado da soma é "+resultado); // concatenamos uma mensagem escrita com uma variável com o sinal de "+"

Se formos fazer com um valor digitado pelo usuário, precisaremos primeiramente importar o pacote de Scanner e criar um objeto, igual ao que já foi mostrado, após isso, faremos a seguinte programação:

int valor1;

double valor2;

System.out.println("Insira o primeiro valor");
valor1 =scan.nextInt() ; // Scanneamos a primeira variável

System.out.println("Insira o segundo valor");
valor2 =scan.nextDouble() ; // Scanneamos a segunda variável

double resultado = valor1+valor2;

System.out.println("O resultado da soma é "+resultado);

Nesse exemplos, fizemos cálculos básicos de soma, mas podemos realizar expressões algébricas extremamente complexas seguindo a mesma lógica.
