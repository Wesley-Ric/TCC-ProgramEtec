Instalação da IDE

Antes de começarmos a realmente aprender Java, precisamos de uma boa IDE de desenvolvimento, isto é, um bom ambiente integrado de desenvolvimento, garantir uma boa IDE no começo facilita todo o aprendizado, pois através dela nós podemos baixar boas extensões, plugins, etc.

Por esses motivos, eu recomendo o NetBeans como IDE, não é das mais leves IDE'S, mas as suas funcionalidades compensam.
