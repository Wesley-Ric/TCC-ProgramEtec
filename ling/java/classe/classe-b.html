Java Classes
O que são classes?
Declaração
Regras

Para declararmos uma classe em Java é algo bem simples, primeiramente precisamos saber se queremos defina-la como privada ou pública, caso seja privada, usaremos o termo private, e caso seja pública, usamos public. Após definirmos isso, digitamos o termo class e entraremos com o nome da classe, que pode ser qualquer um, usaremos como exemplo o nome "PrimeiraClasse". E por fim abriremos chaves, no fim do processo teríamos algo parecido com isso:

public class PrimeiraClasse{

}

Caso queíramos uma classe privada (classe que não pode ser acessada pelas outras), o código ficará da seguinte forma:

private class PrimeiraClasse{

}

E como nos nossos estudos usaremos a command-line da IDE para testes, precisamos do seguinte parâmetro dentro da nossa classe:

public class PrimeiraClasse{

public static void main(Strings[] args) {
}

}
