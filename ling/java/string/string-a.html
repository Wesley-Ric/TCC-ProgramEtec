Strings
Um pouco sobre as Strings
Comparação de String

Como já foi dito anteriormente, as Strings são um tipo de variável um pouco diferente do Int e Double, sua principal diferença é a capacidade de armazenar letras e caracteres especiais, consequentemente, tornando-as variáveis não calculáveis.

As variáveis de tipo String são muito utilizadas para armazenar CPF, que, por mais que sejam números, não são utilizados para nenhum tipo de cálculo, sendo assim, não é necessária a criação de um variável de tipo int.

Além de números não calculáveis, as Strings também são muito utilizadas para armazenar nomes, códigos, endereços, etc.
