Adicionar elementos

Agora que nos já sabemos o que é um vetor e para que ele serve, vamos ver como se declara um em JavaScript.

var amigos = [ "João", "Luiza", "Ronaldo", ] --> Dessa forma estamos colocando essas 3 strings dentro de uma mesma variável, o mesmo poderia ser feito com outros tipos de variáveis.

Com o vetor declarado, nós podemos começar a operar com ele.
