Captar elementos
Por ID
Por Class
querySelector()

Se o seu caso for com classes, agora aprenderemos a captar o elementos através de class, o código em si é basicamente o mesmo, mudando apenas a última palavra.

var idade document.getElementByClass('class-idade') // -> Esse método é usado para coletar o elemento através da CLASS

IMPORTANTE: Esses métodos devem ser usados sempre reverenciando uma variável

No código acima, nos usamos o método para captar através da CLASS do elemento, note que estamos guardando o dado na variável "idade".

  • var - criação da variável

  • idade - nome da variável

  • document - indicando que é um elemento dentro do documento

  • .getElementByClass - método para captar elemento através da class
