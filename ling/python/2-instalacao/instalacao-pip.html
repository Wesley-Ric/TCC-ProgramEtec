Pip install

Você instalou o PyCharm e agora veio aqui e se perguntou:

- O que será esse pip install?

Eu te respondo, pip install é um comando que você usa no terminal do PyCharm para instalar as bibliotecas que não vem de fábrica com o Python e com o PyCharm. O pip por padrão já vem instalado no seu Python, basicamente ele é um gerenciador de pacotes do Python que gerencia as suas bibliotecas as instalando, desinstalando e atualizando. O pip install vai instalar uma biblioteca específica que você queira, lembrando que as bibliotecas só precisam ser instaladas uma única vez, após isto só será necessário importar no seu código se for utilizá-la. Na própia IDE.

Onde usar o pip

Como dito anteriormente o pip não é ultilizado onde nós escrevemos o código Python, para isso usamos o terminar, que pode ser o próprio CMD, porém neste caso irei ensinar a usar o terminal do pycharm, que pode ser acessado como mostra na imagem a baixo:



Então abrirá o terminal:
Com o pip podemos usar diversos comandos não no o "pip install", embora seja o comando que mais iremos ultilizar, pois ele instala as bibliotecas que ultilizamos, porém temos comandos como:
  • pip unistall (nome da biblioteca sem parênteses) - desinstala a biblioteca especificada
  • pip install -upgrade (nome da biblioteca sem parênteses) - atualiza a biblioteca especificada
  • pip list - lista todas as bibliotecas instaladas no Python do seu computador:



Temos também o pip freeze que semelhante ao pip list mostra as bibliotecas instaladas, porém apenas as que foram instaladas manualmente com o pip install:
