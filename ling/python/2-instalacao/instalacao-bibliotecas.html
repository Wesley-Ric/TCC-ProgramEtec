Bibliotecas

Certo, você já instalou o Python, instalou e configurou sua IDE e aprendeu a usar o comando pip, agora precisamos saber o que são bibliotecas, suas funções e as principais bibliotecas no python

O que é uma biblioteca em Python?

As bibliotecas são basicamente um conjunto de módulos e funções em Python, essa resposta pode não fazer sentido agora, e então nos leva a outra pergunta: o que são módulos e funções? Um módulo é um arquivo .py com instruções (que é a mesma coisa que funções) em python.

Eu sei, ainda parece muito confuso, então para elucidar mais sua mente nada melhor que darmos um exemplo:

PI = 3,14

def area(raio):
retrurnPI * (raio** 2)

def comprimento_circulo(raio)
retrurn 2*PI*raio
Nesse exemplo criamos um código que representa um círculo e nosso arquivo se chama circulo.py.

Para usarmos o módulo acima, podemos fazer como mostra abaixo

import circulo #o nome circulo e por conta do nome do arquivo ser circulo.py

print(circulo.PI)
print(circulo.area(5))
print(circulo.comprimento_circulo(5))

Perceba que usamos o comando import circulo para que pudéssemos acessar as definições, ou seja, as funções (def) do nosso programa. De modo geral, essa é a forma pela qual usamos módulos em Python. Caso quiséssemos utilizar apenas a função area() poderíamos importar apenas ela por meio do comando from circulo import area. Isso é útil em caso de módulos muito grandes e quando queremos usar apenas uma ou outra função deles.

Por fim temos que as bibliotecas são programas feitos por outro desenvolvedores python e disponibilizados para que possamos resolver problemas em nosso programa de forma mais eficiente e rápida, por meio das bibliotecas em python podemos expandir seu leque de soluções, podendo fazer a criação de sites e APIs, ciência de dados ,inteligência artificial, vizualização de dados, automações, interfaces gráficas, etc.

Principais bibliotecas no Python

Python tem uma das maiores gamas de bibliotecas no desevolvimento e vem crescendo cada dia mais, dentre essas podemos destacar algumas:

1 - Pandas


Pandas é um biblioteca gratuita, utizada para o tratamento de dados, lançado em 2008. O Pandas oferece diversas estruturas de dados e operações de usar e de alto desempenho para manipular dados na forma de tabelas numéricas e séries temporais.

1 - NumPy


NumPy é uma biblioteca usada para a computação númerica em dados que podem estar na forma de grandes arrays e matrizes multidimensionais. Essas matrizes multidimensionais são os objetos principais em NumPy, onde suas dimensões são chamadas de eixos e o número de eixos é chamado de classificação.

1 - SciPy


SciPy é uma biblioteca livre para a computação científica e computação técnica sobre os dados. Ele foi criado como um projeto de biblioteca comunitária, e inicialmente lançado por volta de 2001. Biblioteca SciPy é construído sobre o objeto de matriz NumPy e é parte da pilha NumPy que também inclui outras bibliotecas de computação científica e ferramentas como Matplotlib, SymPy, pandas etc.

Conclusão

Para terminar, podemos ver a importância e a quantidade de novas funções que as bibliotecas podem nos oferecer, e podemos ver que nós mesmos somos capazes de criar nossos próprios módulos e bibliotecas para usar durante o dia a dia, assim aumentando a já imensa comunidade de Python.
