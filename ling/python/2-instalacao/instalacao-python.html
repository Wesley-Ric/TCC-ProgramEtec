Então você se decidiu que quer estudar Python, está no lugar certo, nesse breve tutorial irá aprender de forma rápida e em tópicos como você pode dar o primeiro passo para a sua jornada python, então vamos, é claro, começar pelo começo.

Instalaçõo do Python

Para instalar o python precisamos primeiro baixar seu instalador em nossa máquina, você pode fazer isso clicando aqui.

passo a passo da instalação

Baixando

O primeiro passo é baixar o Python, você pode acessar o site python.org para fazer o download do arquivo.

Após isso você irá clicar na aba downloads

Segue a imagem:

Depois um menu será expandido na parte "Download for windows" você irá clicar no botão "Python 3.10.8" (que é a versão mais recente até o presente momento 10/2022)

Segue a imagem:

Em seguida seu download se iniciará, assim você poderá abrir o executável clicando duas vezes no ícone no seu navegador.

Segue a imagem:

Logo após isso irá abrir o programa executável do python, para fazer a instalação do mesmo em seu computador com a opção de "Install Now", podemos clicar nessa opção para seguirmos.

Imagem ilustrativa:



Então aparecerá um pop-up do windows, o famoso "deseja permitir que este aplicativo faça alterações no seu computador?", podemos clicar em sim, assim o instalador entrará numa tela com um barra de progresso:




Depois disso O Python já estará instalado no seu computador e aparecerá a seguinte tela:




A partir daqui o Python já está instalado em seu computador, podemos inclusive fazer algumas codficações em seu editor de texto Python, porém com uma IDE é muito melhor, e o que aprenderemos no proximo bloco é como instalar a IDE PyCharm para começar a escrever nossos primeiros códigos em Python.
