Operadores de indentidade

Os operadores de indentidade compara de a "indentidade" do objeto é a mesma de outro, isso é eles comparam se eles ocupam o mesmo espaço na memória, o que significa que se trata do mesmo objeto.

Operador Conceito Exemplo
is Retorna True se as variáveis comparadas forem o mesmo objeto nome is 'Wesley'
is not Verifica se um valor é menor que o outro x is not 'Pyhton'

alguns exemplos para podermos entender melhor:

IS (é)

time_wesley = "Palmeiras"
time_matheus = "Corinthians"
time_italo = "Corinthians"

if time_wesley is time_matheus:
print( "time_wesley e time_matheus = mesmo objeto")
print( "Wesley e Matheus torcem para o mesmo time!\n")

else:
print( "time_wesley e time_matheus = objetos diferentes" )
print( "Wesley e Matheus torcem para times doferemtes!\n" )

if time_matheus is time_italo:
print( "time_matheus e time_italo = mesmo objeto")
print( "Matheus e Italo torcem para o mesmo time!\n")

else:
print( "time_matheus e time_italo = objetos diferentes" )
print( "Wesley e Matheus torcem para times diferentes!\n" )

Resultado:

IS NOT (não é)

time_wesley = "Palmeiras"
time_matheus = "Corinthians"
time_italo = "Corinthians"

if time_wesley is not time_matheus:
print( "time_wesley e time_matheus = objetos diferentes")
print( "Wesley e Matheus torcem para times rivais! \n ")

else:
print( "time_wesley e time_matheus = mesmo objeto" )
print( "Wesley e Matheus torcem para o mesmo time!\n" )

if time_matheus is not time_italo:
print( "time_matheus e time_italo = objetos diferentes")
print( "Matheus e Italo torcem para times rivais!\n")

else:
print( "time_matheus e time_italo = mesmo objeto" )
print( "Wesley e Matheus torcem para o mesmo time!\n" )

Resultado:

Nesses codigos usamos o is para verificar se o valor contido dentro das strings tem o mesmo valor na memoria assim entrando no if e a mesma coisa acontece no is not, poré é comparado se são diferentes para poder entrar no if.
