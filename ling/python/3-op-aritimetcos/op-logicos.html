O que são operadores lógicos?

Os operadores lógicos são símbolos que nos permite contruir algo extremamente útil em python ou em quealquer outra linguagem: os testes lógicos.

Em python temos três operadores lógicos: o and, or e o not.
Operador Definição
and Retorna True se ambas as afirmações forem verdadeiras
or Retorna True se uma sas afirmações forem verdadeiras
not Retorna Falso se o resultado for verdadeiras
Para ficar mais claro podemos mostrar a tabela a seguir:
Tabela AND
Condição 1 Condição 2 Resultado AND
True True True
True False False
False True False
False False False
Como podemos ver mais claramente, com o operador lógico AND apenas será verdadeiro se as duas condições forem verdadeiras dentro da estrutura de decisão.
Tabela OR
Condição 1 Condição 2 Resultado AND
True True True
True False True
False True True
False False False
Já no caso do operador lógico OR ou OU se apenas uma das condições for verdadeira na estrutura if o resultado será verdadeiro, dando false apenas no caso das duas (ou mais) cindições serem falsas.
Tabela NOT
Condição 1 Resultado AND
True False
False True

O operador NOT é o mais fácil de entender pois ele apenas pegara o resultado da operação e irá invertelo, ou seja se der True será False e se for False se tornará True.

veja algumas ultilizações de cada operador:

And (e)

num1 = 7
num2 = 4

if num1 > 3 and num2 < 8:
print("As duas condições são verdadeiras")

Or (ou)

num1 = 6
num2 = 10

if num1 > 4 and num2 < 8: #note que apenas "num1 > 4" é verdadeiro
print("Apenas UMA das condições é verdadeiras")

Resultado
