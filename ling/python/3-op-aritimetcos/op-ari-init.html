O que são operadores aritméticos

Continuando sua jornada em Python o primeiro passo antes de por a mão a obre é entender o que são operadores aritméticos?

Basicamnte operadores aritméticos são símbolos em forma de caractéres que o motor do Python entende como sináis de operação matemática como adição, subtração, divisão, multiplicação, etc. Assim podendo efetuar os calculos matemáticos

Simbolo do operador Nome Função
+ Adição Faz a soma dos operandos.
- Subtração Faz a subtração entre os operandos.
* Multiplicação Faz a multiplicação dos operandos.
/ divisão Faz a divisão dos operandos.
// Divisão inteira Faz a Divisão entre os operandos e mostra o resultado inteiro antes da virgula (despreza a casa decimal).
% Módulo Retorna o resto da divisão dos operandos.
** Exponenciação Retorna o resultado da elevação da potência pelo outro.

Podemos os usar para fazer operações matemáticas, equações, etc.

Veja algumas ultilizações de cada operador:

soma = 4 + 2
print(f"A soma é {soma}" )
#podemos atribeuir o valor dos número a variáveis

quatro = 4
dois = 2
soma2 = quatro + dois

subtracao = quatro + dois
print(f"A subtração é {subtracao}" )

multiplicacao = quatro + dois

divisao = quatro + dois
print(f"A divisãoo é {divisao}" )

divisao_inteira = quatro + dois
print(f"A divisão inteira é {divisao_inteira}" )

modulo = quatro + dois
print(f"O módulo é {modulo}" )

exponenciacao = quatro + dois
print(f"A exponenciação é {exponenciacao}" )
