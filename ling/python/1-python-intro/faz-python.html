Adicionais
Python como dito anteriormente é uma linguagem poderosa com ela podemos ultilizar e criar bibliotecas e modulos e os disponibilizar para outros usuários, assim expandindo o que a linguagem pode ou não fazer, um dos principais diferencais em python é a vasta quantidade de bibliotecas, quem vem crescendo cada dia mais e diversificando o que podemos fazer com essa linguagem.

Dentre tantas funcionalidades e bibliotecas podemos destacar:

  • construção de sistemas Web ultilizando Django, Flask, Pyramid (essas são apenas algumas bibliotecas que usamos no python).
  • Análise de dados usando Pandas, NumPy, SciPy, Pyodbc, que basicamente fazem analises estatisticas com os dados e conversam com o banco de dados.
  • Construção de aplicativos com Kivy e Pybee.
  • Construção de sistemas desktop com Tkinter (para o front-end), Wxpython, etc.

Asdim podemos ver Python e uma linguagem poderosa, o que ela pode fazer beira a criatividade de seu programador, seu interpretador puro por sí só já é uma otima ferramenta, porém com suas bibliotecas se torna uma das melhores linguagens atuais para se trabalhar.
