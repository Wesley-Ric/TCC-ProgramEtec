Python como dito anteriormente, é uma linguagem poderosa, com ela podemos utilizar e criar bibliotecas e módulos, e os disponibilizar para outros usuários, assim expandindo o que a linguagem pode ou não fazer, um dos principais diferenciais em python é a vasta quantidade de bibliotecas, quem vem crescendo cada dia mais e diversificando o que podemos fazer com essa linguagem.

Dentre tantas funcionalidades e bibliotecas podemos destacar:

  • Construção de sistemas Web utilizando Django, Flask, Pyramid (essas são apenas algumas bibliotecas que usamos no python).
  • Análise de dados usando Pandas, NumPy, SciPy, Pyodbc, que basicamente fazem análises estatísticas com os dados e conversam com o banco de dados.
  • Construção de aplicativos com Kivy e Pybee.
  • Construção de sistemas desktop com Tkinter (para o front-end), Wxpython, etc.

Assim podemos ver que Python é uma linguagem poderosa, o que ela pode fazer beira a criatividade de seu programador, seu interpretador puro por sí só já é uma ótima ferramenta, porém com suas bibliotecas se torna uma das melhores linguagens atuais para se trabalhar.
