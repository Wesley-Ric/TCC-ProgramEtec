Python e outras linguagens

  • O Python foi projetado para facilitar a leitura e possui semelhanças com o inglês com influência da matemática.
  • Python usa novas linhas para completar um comando, ao contrário de outras linguagens de programação que geralmente usam ponto e vírgula ou parênteses.
  • Python depende de recuo ou indentação, usando espaço em branco, para definir o escopo; como o escopo de loops, funções e classes. Outras linguagens de programação costumam usar colchetes para essa finalidade.
