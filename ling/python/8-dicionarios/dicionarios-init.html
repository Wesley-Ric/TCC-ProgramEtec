Dicionários Introdução

Os dicionários são coleções de itens e seus elementos são armazenados de forma não ordenada.

Seus elementos contém uma chave e valor, isto é:

  • Uma chave que vai servir para indexar (posicionar) determinado elemento no dicionário.
  • Um valor que contém… Bem, um valor :laughing: Este valor aceita diversos tipos: listas, outros dicionários, inteiros, strings e etc.

    • Sua sintaxe básica é: {'chave': 'valor'}. Utiliza-se {} para delimitar o dicionário e a chave é separada do valor por dois pontos :.

    exemplos:
    dicionario = {'chave':'valor'}

    (type(dicionario))

    Criando dicionários

    Vejamos então as 6 maneiras possiveis de se criar um dicionário

    Para começarmos vamos com o modo mais simples de se criar um dicionário:

    dicionario = {'primeiro':1, 'segundo':2, 'terceiro':3}

    Podemos tambem usar o dict no python, passando as chaves e os valores como parâmetros :

    dicionario2 = (primeiro=1,segundo=2,terceiro=3 )

    Utilizando zip para concatenarmos a chave:valor em um objeto dict:

    dicionario3 = dict(zip(['primeiro', 'segundo', 'terceiro'], [1, 2, 3]))

    Com uma lista de tuplas:

    dicionario4 = dict([('primeiro', 1), ('segundo', 2), ('terceiro', 3)])

    Com Dict Comprehensions:

    dicionario5 = {name: idx + 1for idx, name inenumerate(('primeiro', 'segundo', 'terceiro'))}

    Por mais que pareça estranho, da pra transformar uma variável dicionário em um tipo dicionário:

    dicionario6 = dict({'primeiro': 1, 'segundo': 2, 'terceiro': 3 })
