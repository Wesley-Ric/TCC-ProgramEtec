Estrutura de repetição for

As estruturas de repetição são recursos das linguagens de programação responsáveis por executar um bloco de código repetidamente através de determinadas condições especificas.

O Python contém dois tipos de estruturas de repetição: for e while.

Nesta página vamos ver o for

For (para)

O for é utilizado para percorrer ou repetir uma sequência de dados, ou seja, uma sequência de códigos seja esse uma lista, tupla, string, executando assim um conjunto de instruções a cada item.

Como você já sabe, Python utiliza identação para separar blocos de código: nos loops utilizando for não é diferente.

A sintaxe do for é basicamente: for < nome da variável> in < iterável>

vamos entender melhor:

  • < nome da variável> é o nome da variável que vai receber os elementos de < iterável>
  • < iterável> é o conteiner de dados sobre o qual vamos iterar, podendo ser: uma lista, tupla, string, dicionário, etc.

    • Para entender melhor o for, vamos criar um contador que conte de 0 a 6, para isso, vamos criar uma variável contador com o valor de 100, depois disso para cada unidade no contador iremos printar ele na tela, logo depois vamos sair do loop printando na tela que a conta terminou.

    contador = 6

    for unidade in range(contador)
    print(unidade)
    else:
    print( "Contamos tudo")

    Nesse caso foi utilizado o range() para usar variável como um escopo para o contador parar, isso normalmente é utilizado quando usamos números inteiros.
