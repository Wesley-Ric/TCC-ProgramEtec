Funções

A função range é de grande ajuda quando o tema é repetição, laços, for etc. Ela básicamente funciona como uma "distÂncia" até onde o código vai de acordo com o número que insere.

Ela gera uma sequência de números pela qual podemos iterar!

  • Podemos indicar nela:
  • O inicio de uma sequência.
  • O passo ou de quantos em quantos ela vai ir.
  • O valor final da sequência.
    • Com isso temos uma sequência de números para ultilizarmos.

    Podemos usar o range com três escopos, ínicio, parada e passo (intervalo)

    range(parada)

    range(inicio, parada)

    range(inicio, parada, passo(intervalo))

    Exemplo de uso:

    Range parada

    for num in range(5)
    print(num)

    Saída:


    Como podemos ver ele conta até 5, ou seja o 5 é o stop ou a parada.

    Range início e parada

    for num in range(3 , 4)
    print(num)

    Saída:


    No segundo caso temos que vamos começar no 3 e terminar no 5 então ele só contará o 3 e o 4

    Range início, parada e pulo (intervalo)

    for num in range(3 , 5, 2)
    print(num)

    Saída:


    Note que nesse último caso o inicio é o número 1, o final é o 5, ou seja ela termina no 4 que é um antes do 5, e o intervalo é 2, então ele conta de dois em dois tendo 1 + 2 = 3, printando o 1 e 3.

    Funcão enumerate()

    Uma boa dica para seus códigos em Python para se usar com o for, é a função enumerate().

    Ela funciona como se fosse um contador que você cria com uma variável porém é imbutido no próprio for!

    Podemos então ao invés disso:
    contador = 0
    computador = ['Processador', 'Teclado', 'Mouse' ]
    for elemento in computador:
    print(f"Índice = {contador} | Valor={elemento}")
    contador += 1

    Saída:

    Jogamos o contador fora e fazemos isso:

    computador = ['Processador', 'Teclado', 'Mouse' ]
    for indice, valor in enumerate(computador):
    print(f"Índice = {indice} | Valor={valor}")

    Saída:


    Esse código é o que podemos chamar de Pythonico, pois ele segue as melhores praticas da programação em Python, é menor e usa menos memória.

    Em loops com while a lógica é a mesma. O continue irá finalizar o loop atual, iniciando novamente no início do while.

    Outra maneira de iterar sobre os índices, é combinar as funções range() e len().

    A função len() retorna o tamanho de um iterável (lista, tupla, set).

    Podemos combiná-los da seguinte forma:

    computador = ['Processador', 'Teclado', 'Mouse' ]
    for indice, indice in range(len(computador)):
    print(f"Índice = {indice} | Valor={valor}")
