Funções

A função range é de grande ajuda quando o tema é repetição, laços, for etc. Ela básicamente funciona como uma "distÂncia" até onde o código vai de acordo com o número que insere.

Ela gera uma sequência de números pela qual podemos iterar!

  • Podemos indicar nela:
  • O inicio de uma sequência.
  • O passo ou de quantos em quantos ela vai ir.
  • O valor final da sequência.
    • Com isso temos uma sequência de números para ultilizarmos.

    Podemos usar o range com três escopos, ínicio, parada e passo (intervalo)

    range(parada)

    range(inicio, parada)

    range(inicio, parada, passo(intervalo))

    Exemplo de uso:

    Range parada

    for num in range(5)
    print(num)

    Saída:


    Como podemos ver ele conta até 5, ou seja o 5 é o stop ou a parada.

    Range início e parada

    for num in range(3 , 4)
    print(num)

    Saída:


    No segundo caso temos que vamos começar no 3 e terminar no 5 então ele só contará o 3 e o 4

    Range início, parada e pulo (intervalo)

    for num in range(3 , 5, 2)
    print(num)

    Saída:


    Note que nesse último caso o inicio é o número 1, o final é o 5, ou seja ela termina no 4 que é um antes do 5, e o intervalo é 2, então ele conta de dois em dois tendo 1 + 2 = 3, printando o 1 e 3.

    Funcão enumerate()

    num = 0
    while num < 5
    num += 1
    if num == 3:
    break

    print(num)

    Saída:

    Quando a variável atribuir o valor 3 o laço é finalizado pelo break, encerrando o loop.

    Continue

    Funciona de maneira similar ao break, contudo ao invés de encerrar o loop ele pula todo código que estiver abaixo dele (dentro do loop) partindo para a próxima iteração.

    Vamos ao exemplo:

    for num in range(5)
    if num == 3:
    print("Encontrei o 3")
    continue
    else:
    print(num)

    print("Estou abaixo do IF")

    Saída:


    Note que quando o a condição entra no IF ele não printa nem o num nem o "estou abaixo do IF" ignorando o restande do bloco de código.

    Em loops com while a lógica é a mesma. O continue irá finalizar o loop atual, iniciando novamente no início do while.

    Veja o exemplo:

    num = 0
    while num < 5:
    num += 1

    if num == 3:
    continue

    print(num)

    Saída:


    O resultado desse código é que o 3 não apareça, pois o print() que imprime os números está abaixo do continue.

    Pass

    O pass nada mais é que uma forma de fazer um código que não realiza operação nenhuma.

    Mas calma, ele tem uma razão de existir no Python!

    Como os escopos de Classes, Funções, If/Else e loops for/while são definidos pela indentação do código (e não por chaves {} como geralmente se vê em outras linguagens de programação), usamos o pass para dizer ao Python que o bloco de código está vazio.

    Veja alguns exemplos:

    for item in range(5):
    pass

    while False:
    pass

    classclasse:
    pass

    if True:
    pass
    else:
    pass

    def funcao():
    pass

    Saída:


    Caso não utilizemos o pass, veja o que acontece:

    classclasse:
    def funcao():
    pass

    Saída:


    Isso acontece pois o Python entende que as próximas linhas de código fazem parte do mesmo escopo, mas como não estão indentadas um erro IndentationError é lançado.
