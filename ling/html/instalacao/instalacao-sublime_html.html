Como instalar a Sublime Text

Além do Vs Code, podemos usar também o Sublime Text, que terá as mesmas funções que o Visual Studio Code, os dois programas serão utilizados para fazer diversas linguagens como html, css e java.

Sublime Text

Sublime Text assim como o Vs code, também é um código fonte multiplataforma e shareware com uma interface de programação de aplicativos em linguagem de Python, podendo ser usadas nas linguagens de programação e linguagens de marcação, funções podem ser adicionadas por usuários com plug-ins, geralmente criados pela comunidade e mantidos sob licenças de software livre.

Para instalar o Sublime Text, basta clicar aqui e seguir o passo a passo abaixo:

Primeiramente iremos acessar a página do Sublime Text e Clicar no botão Download; após clicar neste botão ele te abrirá uma nova página, conforme a imagem abaixo:

Você escolherá de acordo com seu Software, teremos as opções para Windows, Linux e Mac, neste caso estamos fazendo a instalação no Windows, mas todos os Softwares serão o mesmo método de isntalção.

A partir de agora, ele já terá baixado em seu computador, e ficará com a ".exe" no canto inferior esquerdo igual a imagem abaixo;

Após ser feito o download, abra a pasta onde foi baixado o ".exe", para que possa instalar em seu computador;

Após abri o ".exe" você mudará a opção de "Eu não aceito o acordo" para "Eu aceito o acordo";

Após este passo a passo clique no Botão "Instalar";

Se caso preferir usar seu programa que já tem instalado em sua máquina ou este que iremos ensinar a instalar pode, fica de sua preferência.s
