Tags de textos

As tags de texto são utilizadas para as escolhas de textos, quebra de linhas, páragrafos, entre outras coisas que podemos fazer, para isso veremos algumas que são importantes.

Para começar vamos falar sobre a tag "p", aue tem como sua função separar os textos em páragrafos.

Utilizamos também as tags "span" que tem funcionalidade e características parecidas com os parágrafos, elas costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem

Utilizamos as tags "i" e "br" para dar estilo pu destaque para alguma palavra ou frase, a tag "i" deixa em italico, já a tag "br" tem como função deixar a parte sugerida em negrito. Lembrando sempre que as tags podem ser utlizadas juntas.

<! DOCTYPE html >
< html lang= "en">
<head >
</head>
<p > Olá ProgramEtec</p >
<br >ProgramEtec uma iniciativa estudantil para ensinar programação a qualquer pessoa, seja onde for não importa qual momento ou dispositivo.</br >
<i >ProgramEtec aprenda em qualquer momento ou lugar.</i >
</html>
</span>

Tags de Títulos

As tags de título possuem valor semântico, variando entre seis níveis hierárquicos. Para fins de otimização SEO é importante entender como funcionam, e fazer uma utilização adequada.

As tags se tem inicio em "h1" onde deixa seu título em destaque, e termina em "h6" cada tag tem seu valor semantico menor, de ordem crescente. Quando vamos colocar um título podemos coloca-los de formas adequedas e de fácil visualização podendo utilizar as tags existem.

<! DOCTYPE html >
< html lang= "en">
<head >
</head>
<title > Olá ProgramEtec</title >
<h1 >Precisa estar desta forma, em qualqer tag de h1 a h6. </h1 >
</html>
</span>
