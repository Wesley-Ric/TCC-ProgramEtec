Tags de Listas

Dentro de uma página precisamos muitas vezes colocar algmas listas, seja uma página de receita, uma página de escola,ou para fazer um menu, para isso ensinaremos a vocês agora algumas tags para utilizar neste casos.

As listas se iniciam através da tag "ul" que te da a opção de fazer listas de várias ordens, mas todas nas verticais.

Dentro de cada "ul" estão localizadas as tags "li" que são as que colocam em listas suas escritas. Conforme podemos ver no exemplo abaixo:

<! DOCTYPE html >
< html lang= "en">
<ul >
<li >Nome </li >
<li >Telefone </li >
<li >Endereço </li >
<li >País </li >
</ul >
</html >
