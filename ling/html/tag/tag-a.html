Chegou a hora que todos nós esperávamos, o momento de começarmos nossa Primeira Página, mas se você está aqui e está precisando de algumas dicas não se preocupe, pois aqui você encontrará tudo o que precisar.

O que é uma Tag?

É um elemento do Html, todas as marcações se iniciam da Tag, seja ela um texto, uma mídia(vídeos, textos, imagens), também através das tags podemos marcar para não se perder dentro de uma programação.

Vamos começar falando sobre as principais Tags do html e para que elas servem, quase todas as tags são utilizadas com a letra minúscula.

Iremos iniciar falando sobre a Tag "html"

A tag Html será utilizada para envolver todo o documento html e será utilizada da seguinte forma:

<'!DOCTYPE html>
<'html lang="en">
<'/html>

Também temos a tag "head". Utilizada normalmente para o cabeçalho da sua página, para colocar a logo da sua empresa ou trabalho.

A tag "head" fica dentro da tag "html", para que comece a criar uma interface na sua página.

<'!DOCTYPE html>
<'html lang="en">
<'head>
<'/head>
<'title>Olá ProgramEtec<'/title>
<'/html>

A tag "style" dará o estilo para sua página é dentro delas que iremos colocar, cores e class. A partir da tag "style" tudo poderá mudar, então faça bem suas escolhas e vamos para a prática.

<'!DOCTYPE html>
<'html lang="en">
<'head>
<'style>
h1 {color:red;}
p {color:blue;}
<'/style>
<'/head>
<'title>Olá ProgramEtec<'/title>
<'/html>

Tag "meta", fornece todas as informações geradas dentro do seu documento Html.

<'!DOCTYPE html>
<'html lang="en">
<'head>
<'meta charset="UTF-8">
<'meta name="description" content="Free Web tutorials">
<'meta name="keywords" content="HTML,CSS,XML,JavaScript">
<'meta name="author" content="John Doe">
<'meta name="viewport" content="width=device-width, initial-scale=1.0">

<'/head>
<'/html>

Antes de iniciarmos precisamos dar o título para nossa página, aquele nome que fica marcado na janela das páginas, e para isso utlizamos a tag "title".

<'!DOCTYPE html>
<'html lang="en">
<'head>
<'title>HTML Elements Reference<'/title>
<'/head>
<'body>
<'/body>
<'/html>


Essas são somentes as principais vamos conhecer mais tags que são utilizadas no html.

Observação: Todas as tags que são utilizadas não se utilizam Aspas, são imagens ilustrativas, fora isso tudo é utilizado da mesma forma que foi explicado.
